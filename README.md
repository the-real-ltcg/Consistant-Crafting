# Consistant-Crafting
Consistent Crafting: Fixes inconsistent vanilla crafting yields for sensible 1-block-to-1-craft ratios. All recipes in this pack are shapeless/single-ingredient where possible, so items go in any slot. 1 block → 2 slabs, for every vanilla block that has a slab counterpart. 3 Wheat → 3 Bread. 3 Iron Ingots → 1 Bucket.
